==Стили==
1. Для написания inline стилей необходимо атрибуту style элемента присвоить объект, который внутри будет содержать еще один объект
ключами в качестве css свойств и значениями в качестве значений соответствующих css свойств. Имя css свойства можно записывать как
в кавычках (тогда название строки будет полностью совпадать с изначальным названием свойства), так и без них (имя свойства должно быть
записано в виде camelCase). Значение свойства пишется в виде строки (значение в пикселях таже можно записывать в виде числового значения).
Объект со стилями можно присваивать какой-либо переменной вне элемента для повышения читаемости кода
2. Стили рекомендуется выносить в отдельный .css файл. Чтобы его подключить использовать стандартный ES6 синтаксис подключения файлов:
    import './Style.css';


==SASS (SCSS)==
npm install node-sass
После установки пакета достаточно переименовать .css файлы в .scss и подключить их внутри компонента


==CSS( (SCSS) модули==
1. Переименовать .css (.scss) файл в <название_файла>.module.css(.scss)
2. Внутри компонента импортируем переменнут из данного файла. Например:
import classes form "./Comp.module.scss"
Данная переменная содержит объект, который содержит ключи, совпадающие с названиями классов из .css (.scss) файла.
3. Внутри компонента атрибуту className присваиваем объект со значением classes['название_класса']. Например:
<div className={classes.App}>
    <h2 className{classes['App-header']}>React component</h2>
</div>
