==Создание проекта==
npm install -g create-react-app
create-react-app <project name>


==Структура проект==
package-lock.json: используется для блокировки зависимостей от определённого номера версии
package.json: используется так же для определения списка зависимостей, свойств проекта, описания,
информации об авторе и лицензии, сценариев
.gitignore: указывает список файлов и папок которые не должны попасть в систему контроля версии git
public:
    index.html: входная страница для приложения
    manifest.json: определяет базовые метаданные о расширении, такие как имя и версия,
    некоторые аспекты функциональности (такие, как фоновые скрипты, контент скрипты и
    действия браузера).
src:
    index.js: стартовый файл проекта
    App.js: основной компонент
