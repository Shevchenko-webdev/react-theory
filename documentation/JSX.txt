==JSX==
1. Обязательно импортировать React
2. Должен возвращаться один родительский элемент (количество детей не ограничено)
3. Классы записываются используя ключевое слово className (не class, как в обычном html)
4. Для записи inline стилей внутрь ключевому слову style необходимо присвоить объект внутри которого будет еще
один объект с соответствующими стилями (записывать в виде camelCase
5. Для простых JS конструкций (вызов функций, подсчёт т.п.) можно использовать фигурные скобки {}
6. Для передачи значений внутрь компонента можно использовать конструкцию атрибут+значение (как в html)
Внутрь компонента передаём параметр props и с помощью {} выводим значнение props.<название атрибута>
7. Внутрь тега компонента можно положить другие теги. Для их вывода внутри компонента нужно
использовать props.children
